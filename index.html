Quick Start

  1. Pick a creator mode at the top: Card, Leaflet, or Board.
  2. Add title and description in the left panel.
  3. Upload artwork and tune layers.
  4. Export PNG/PDF or save JSON for reuse.

How To Use

Use the left panel to edit card properties, artwork, and layers. The right preview updates live.

Use deck tools in the header to create decks, save cards, and batch import names.

Open Print Sheet for final page layouts and export PNG/PDF for production.

Shortcuts: Ctrl+S save, Ctrl+Z undo, Ctrl+Shift+Z redo.

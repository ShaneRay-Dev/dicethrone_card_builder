# Dice Throne Card Builder - Project Files

This directory contains all assets and data for the card builder application.

## images/
- Store card art templates and decorative elements
- Supported formats: PNG, JPG, WebP
- Recommended sizes: Card art (600x800px)

## fonts/
- Store custom fonts for card rendering
- Supported formats: TTF, OTF, WOFF, WOFF2

## data/
- Sample cards in JSON format
- Card templates and presets
- Can be imported directly into the builder

## Usage

Place your custom assets in the appropriate folders and reference them in the card data or through the UI.

# Dice Throne Creator - Start Here

A quick guide to using the card builder.

## Quick Start

1. Open `index.html` in a modern browser.
2. Use the editor panel to set card type, phase, name, and description.
3. Upload art or choose a default art option.
4. Adjust layers in the Elements list.
5. Save or export when ready.

## Key Features

- Live preview with layered card rendering
- Art upload + default art selection
- Drag and scale artwork directly on the card
- Layer toggles (bleed, backgrounds, frames, panels, text, cost badge)
- Save/load JSON and export PNG (600 DPI)
- Auto-save and undo/redo
